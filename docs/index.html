Selection to Google Calendar

웹페이지에서 날짜 텍스트를 드래그로 선택하면 Google Calendar에 빠르게 일정을 저장할 수 있는 Chrome 확장 프로그램입니다.

기능 설명

  • 날짜 텍스트 드래그 선택 시 자동 인식
  • 제목 / 시간 / 메모 입력 후 Google Calendar 저장
  • 시간 미입력 시 올데이 일정으로 저장
  • OAuth 로그인 상태 자동 처리

지원하는 날짜 형식

  • yyyy년 mm월 dd일, yy년 mm월 dd일
  • yyyy-mm-dd / yyyy.mm.dd / yyyy/mm/dd
  • yy-mm-dd / yy.mm.dd / yy/mm/dd
  • mm/dd / mm-dd / mm.dd

개인정보 처리방침 (Privacy Policy)

본 확장 프로그램은 Google Calendar 일정 생성을 위해 필요 최소한의 정보만 처리합니다.

수집 및 이용

  • 사용자가 선택한 날짜 텍스트와 입력한 제목 / 시간 / 메모는 Google Calendar API를 통해 일정 생성에만 사용됩니다.
  • 위 정보는 일정 생성 목적 외에 저장하거나 전송하지 않습니다.

보관 및 공유

  • 확장 프로그램은 사용자 데이터를 별도의 서버에 저장하지 않습니다.
  • 제3자에게 개인정보를 공유하지 않습니다.

권한

  • Google Calendar API 사용을 위한 OAuth 인증이 필요합니다.
  • 확장 기능 수행에 필요한 최소 권한만 요청합니다.
