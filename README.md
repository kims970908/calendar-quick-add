# Selection to Google Calendar

웹페이지에서 날짜 텍스트를 드래그로 선택하면 Google Calendar에 빠르게 일정을 저장하는 Chrome 확장 프로그램입니다.

## 기능 설명
- 날짜 텍스트 드래그 선택 시 자동 인식
- 제목/시간/메모 입력 후 Google Calendar 저장
- 시간 미입력 시 올데이 일정으로 저장
- OAuth 로그인 상태 자동 처리

지원하는 날짜 형식
- `yyyy년 mm월 dd일` 또는 `yy년 mm월 dd일`
- `yyyy-mm-dd` / `yyyy.mm.dd` / `yyyy/mm/dd`
- `yy-mm-dd` / `yy.mm.dd` / `yy/mm/dd`
- `mm/dd` / `mm-dd` / `mm.dd`

## 개인정보 처리방침 (Privacy Policy)
본 확장 프로그램은 Google Calendar 일정 생성에 필요한 최소한의 정보만 처리합니다.

수집 및 이용
- 사용자가 선택한 날짜 텍스트, 입력한 제목/시간/메모를 Google Calendar API로 전송하여 일정 생성에 사용합니다.
- 위 정보는 일정 생성 목적 외에 저장/전송하지 않습니다.

보관 및 공유
- 확장 프로그램은 선택 텍스트/입력 정보를 서버에 저장하지 않습니다.
- 제3자에게 데이터를 공유하지 않습니다.

권한
- Google Calendar API 사용을 위한 OAuth 인증이 필요합니다.
- 확장 기능 수행에 필요한 최소 권한만 요청합니다.

## 서비스 이용약관 (Terms of Service)
- 본 확장 프로그램은 사용자의 요청에 따라 Google Calendar에 이벤트를 생성합니다.
- 사용자는 입력한 정보의 정확성에 책임이 있습니다.
- Google 계정 인증이 필요하며, 인증 거부 시 기능이 제한될 수 있습니다.
- 서비스는 현 상태로 제공되며, 일정 생성 실패 등으로 인한 손해에 대해 책임을 지지 않습니다.
